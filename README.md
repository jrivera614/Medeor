# Medeor

**Latin: "I heal, I remedy"**

Interactive Tactical Combat Casualty Care (TCCC), Combat Lifesaver (CLS), and Prolonged Field Care (PFC) training platform.

## Features

- **Training Modules** — MARCH, E-PAWS-B, RAVINES, Hemorrhage Control, Airway Management, PFC Scenarios with branching decision trees
- **Clinical Practice Guidelines** — Full JTS/CoTCCC CPG reference library, searchable
- **Skills Video Library** — 70+ skill videos organized by TCCC CLS curriculum module, linked to Deployed Medicine
- **Ranger Medic Handbook** — Complete field reference with protocols, TEMPs, pharmacology, medical ops, and packing lists

## Stack

- Next.js 14
- React 18
- Deployed on Vercel
- PWA-ready (installable on iOS/Android)

## Deploy

```bash
npm install
npm run dev
```

Push to GitHub, connect repo to Vercel, done.
